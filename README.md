## Setting up

##### Clone the repo

```
$ git clone https://github.com/JaeDukSeo/Only_Numpy_Basic
$ cd Only_Numpy_Basic
```

##### Choose the NN type you wish to run and go into repo

```
$ cd 2_Decoupled_Neural_Network
```

##### run the code in ptyhon3

```
$ python3 1_Red_Box_Case
```
